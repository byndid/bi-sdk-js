To get started:

```
git clone git@github.com:gobeyondidentity/bi-sdk-js.git
yarn example
```

For development:

```
git clone git@github.com:gobeyondidentity/bi-sdk-js.git
cd example
cp .env.local.example .env.local
yarn dev
```
